## 🎯 Publishing Instructions

### Prerequisites
- npm account (https://www.npmjs.com)
- npm CLI installed
- Authenticated with npm

### Steps to Publish

1. **Login to NPM**
   ```bash
   npm login
   ```

2. **Update Version (if needed)**
   ```bash
   npm version patch  # or minor, major
   ```

3. **Publish Package**
   ```bash
   npm publish
   ```

4. **Verify Publication**
   ```bash
   npm info ios-ui-kit
   ```

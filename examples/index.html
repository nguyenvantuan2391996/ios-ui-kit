What is iOS UI Kit?
A pixel-perfect CSS component library following Apple Human Interface Guidelines. Every component matches native iOS — colors, typography, spacing, and animations.
How do I install it?
Link the compiled CSS file in your HTML head: <link rel="stylesheet" href="ios-ui-kit.css">. Then apply the component class names to your HTML elements.
Does it support dark mode?
Yes! All colors use CSS custom properties that map to iOS system colors. When you add data-theme="dark" to your root element, the palette automatically switches.

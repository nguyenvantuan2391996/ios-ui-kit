iOS Kit × Tailwind

Integration Examples

Tailwind utilities (prefixed tw-) work alongside the existing iOS UI Kit classes with zero conflicts.

Example 1 — Button Group

▸ Before (kit only)

<div style="display:flex;gap:8px">
  <button class="btn">Primary</button>
  <button class="btn btn--secondary">Secondary</button>
</div>

▸ After (kit + Tailwind)

<div class="tw-flex tw-flex-wrap tw-gap-2">
  <button class="btn tw-flex-1 tw-sm:flex-none">Primary</button>
  <button class="btn btn--secondary tw-flex-1 tw-sm:flex-none">Secondary</button>
  <button class="btn btn--ghost tw-flex-1 tw-sm:flex-none">Ghost</button>
</div>

<!-- Core visual style stays in .btn / .btn--* (iOS kit).          -->
<!-- Tailwind handles: flex layout, gap, responsive width.         -->

Example 3 — Form Layout

▸ Before (kit only)

<div class="form-group" style="margin-bottom:12px">
  <label class="form-label">First Name</label>
  <input class="input" />
</div>

▸ After (kit + Tailwind layout)

<!-- Tailwind: 1→2 column grid, uniform gap, no inline styles -->
<div class="tw-grid tw-grid-cols-1 tw-md:grid-cols-2 tw-gap-3 tw-mb-4">
  <div class="form-group">
    <label class="form-label">First Name</label>
    <input class="input" />          <!-- iOS Kit: full input styling  -->
  </div>
  <div class="form-group">
    <label class="form-label">Last Name</label>
    <input class="input" />
  </div>
  <!-- Full-width field spanning both columns                   -->
  <div class="form-group tw-md:col-span-2">
    <label class="form-label">Email</label>
    <input class="input" />
  </div>
</div>

<!-- Tailwind: flex row, right-aligned action buttons         -->
<div class="tw-flex tw-gap-2 tw-justify-end">
  <button class="btn btn--ghost">Cancel</button>
  <button class="btn">Save Changes</button>
</div>

Quick Reference — What goes where

iOS UI Kit classes

  • Component visuals → .btn, .card, .input
  • Color & typography tokens
  • States: hover, focus, disabled
  • Dark-mode via CSS custom properties
  • iOS-specific look & feel

Tailwind utilities (tw-*)

  • Layout → tw-flex, tw-grid, tw-gap-*
  • Spacing → tw-mt-*, tw-px-*, tw-py-*
  • Responsive → tw-md:grid-cols-2
  • Alignment → tw-items-center, tw-justify-between
  • Effects → tw-backdrop-blur, tw-opacity-*
ios-ui-kit × Tailwind CSS — additive integration, zero visual regression
